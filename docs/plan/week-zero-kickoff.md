Use this plan before your first full week of study.

## Outcome in 45 Minutes

By the end of this kickoff, you will have:

- One philosopher to start with
- One book/edition selected
- One weekly checklist ready
- One short reflection template ready

## 5-Step Kickoff

1. Read one profile in `docs/authors.md` (10 min).
2. Pick one accessible text in `docs/books.md` (10 min).
3. Copy the block in `docs/templates/weekly-study-checklist.md` (5 min).
4. Open `docs/cheatsheets/argument-analysis-cheat-sheet.md` (10 min).
5. Write your first 5-line reflection in `docs/reviews/core-book-reviews.md` (10 min).

## Starter Routes

- Ethics route: Confucius -> Epictetus -> Marcus Aurelius
- Logic route: Aristotle -> Frege -> Russell
- Politics route: Plato -> Hobbes -> Rousseau -> Mill
- Critique route: Hume -> Marx -> Nietzsche

## First-Week Rule

Keep it small and consistent:

- Read 20-30 pages per session
- Write one argument map per week
- End each week with one reflection paragraph
